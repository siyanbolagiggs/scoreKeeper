# scoreKeeper
This contains the code of a score keeper app for a poker game, you can set a number which determins the winner, the numbers change colour when there is a winner and the buttons get disabled until you hit the reset button.
It contains the JS, HTML and CSS codes for a score keeper app. to open the app,, you can simply create a new folder in your code editor, copy the javaScript, HTML and CSS codes into the folder in their respective ".js",".html" and ".css"files, save it and open the index file from the folder you created.
